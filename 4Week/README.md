- 파일 복사 : `$cp`

- 파일 이동 : `$mv`
  - `$mv 파일 디렉터리` => 디렉터리로 이동
  - `$mv 파일 변경할 파일명` => 파일명 변경

- 파일 삭제 : `$rm`

- 링크 : `$ln`
  - `$cp`는 같은 내용의 다른 파일을 만든다면 `$ln`는 내용을 공유하는 다른 파일을 만드는 것
